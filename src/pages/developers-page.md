# Welcome to Developers


## Roadmap

### Upcoming Minor Releases
The IvorySQL project aims to follow a predefined schedule, releasing at least one minor version per quarter for the first three quarters of the year. If additional releases are needed due to critical bug fixes or security issues, they may occur between these dates. Therefore, this list should be considered a minimum release frequency. On each scheduled date, a new minor version will be released for all supported versions.

The current schedule for upcoming releases is as follows:

- January 2025: Release v4.2 (PG Kernel is 17.2) ```Features: Support for multiple CPU architectures and operating systems; added architecture-related content in documentation.```
- April 2025: Release v4.3 (PG Kernel is 17.3) ```Features: Oracle-compatible auto-increment columns; IvorySQL Operator kernel upgrade.```
- July 2025: Release v4.4 (PG Kernel is 17.4) ```Features: IvorySQL Cloud V4.```

### Next Major Release
The next major version of IvorySQL is planned to be v5.0, scheduled for release in October 2025.

October 2025: Release v5.0 (PG Kernel is 18.0) ```Features: Oracle-compatible rowid.```
