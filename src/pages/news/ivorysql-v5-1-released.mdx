        <p>
          IvorySQL 5.1 is built based on PostgreSQL 18.1, including multiple bug fixes and functional improvements. For the complete update list, please refer to the official documentation site.
        </p>

        <h2 id="pg-features">Enhanced Features from PostgreSQL 18.1</h2>
        <ul>
          <li>Check the CREATE permission for Schemas in CREATE STATISTICS.</li>
          <li>Avoid potential integer overflows in size allocation calculations in libpq.</li>
          <li>Prevent the "unrecognized node type" error when the DEFAULT clause of SQL/JSON functions (e.g., JSON_VALUE) contains COLLATE expressions.</li>
          <li>Avoid incorrect optimization of constant HAVING clauses that include grouping sets.</li>
          <li>Disable parallel execution in hash right semi joins.</li>
          <li>Avoid potential division-by-zero errors when generating ordered-append execution plans.</li>
          <li>Fix the issue where the query planner might fail for index types that support ordered access but not index-only scans.</li>
        </ul>
        <p>For more details, see the official <a href="https://www.postgresql.org/docs/release/18.1/" target="_blank" rel="noopener noreferrer">PostgreSQL 18.1 Release Notes</a>.</p>

        <h2 id="ivory-features">New Features of IvorySQL 5.1</h2>
        <ul>
          <li><strong>Upgraded to PG 18.1 Kernel</strong></li>
          <li><strong>Online Experience:</strong> IvorySQL 5.1 launches a browser-based interactive experience environment, allowing users to explore and evaluate its features in real time without local installation.</li>
          <li><strong>Full-Platform Installation Package Support:</strong> Provide multi-platform installation packages covering X86, ARM, MIPS and LoongArch architectures.</li>
          <li><strong>Containerized Deployment Support:</strong> Enable one-click deployment of IvorySQL 5.1 standalone instances or high-availability clusters via Docker Compose, Podman, Swarm, Helm and Operator tools.</li>
          <li><strong>Synchronized Release of IvorySQL Cloud 5.1:</strong> Integrate the IvorySQL 5.1 database and its surrounding ecosystem to realize visual full-lifecycle management including database provisioning and deprovisioning, monitoring and operation.</li>
          <li><strong>PostgreSQL Extension Support:</strong> Add support for 10 PostgreSQL extensions, namely pg_cron, pgAudit, PostGIS, pgRouting, PGroonga, ddlx, pgsql-http, system_stats, plpgsql_check and pgvector.</li>
        </ul>

        <h2 id="fixed-issues">Fixed Issues</h2>
        <ul>
          <li>Clean up warning messages during the compilation process.</li>
          <li>Fix the issue where the PL/iSQL parser rejects parenthesized syntax in SELECT INTO expressions.</li>
          <li>Correct the documentation description about the missing uuid-ossp extension during source code installation.</li>
        </ul>
