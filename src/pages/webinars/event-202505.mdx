    <div className="tech-card">
      <h4>Project 1: Add One-click Installation Script for IvorySQL</h4>
      <div className="content-text">

      * **Selected Student**: Wang Shuo
      * **Project Mentor**: Jiao Shuntian
      * **Technical Field**: PostgreSQL
      * **Project Details**: [View Here](https://summer-ospp.ac.cn/org/prodetail/258430417)
    </div>

    <div className="tech-card">
      <h4>Project 2: Develop sys_guid Function Based on uuid-ossp for IvorySQL</h4>
      <div className="content-text">
        IvorySQL currently requires a `sys_guid` function that ensures database-level uniqueness to support Oracle compatibility features. This project plans to develop this function based on the `uuid-ossp/e2fs` extension.
      </div>
      * **Selected Student**: Chong Penghao
      * **Project Mentor**: Niu Shiji
      * **Technical Field**: PostgreSQL
